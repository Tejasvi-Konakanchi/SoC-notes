# Introduction to Digital Electronics

<b>What is a Signal ? </b>
<br>
A signal is a expresses information about a physical quantity with respect to any parameter (usually time).

In electronics, usually signal is a variation of electrical current (I).

- If 'I' is the same for different values of time, then it is called <i>Direct Current (DC). </i>
- If 'I' is changing with respect to time, then it is called <i> Alternating Current (AC). </i>
</br>

<b> Analog vs. Digital Signals? </b>
<br>
| Feature | Analog Signal | Digital Signal |
|---------|---------------|----------------|
| Definition | A continuous time signal is a signal that is defined at every moment in time. | A discrete-time signal is a signal that is defined only at specific points in time, rather than at every instant. |
| Accuracy | Limited by noise and component tolerances | High accuracy due to discrete levels |
| Storage/Processing | More difficult | Easy using computers and digital circuits |
| Examples | Microphone output, ECG, temperature sensor, AM/FM radio | Binary data, USB signals, Ethernet, microcontroller outputs |
| Typical Values | Infinite possible values | Finite values (usually 0 and 1) |
</br>

<b> Introduction to Digital Electronics: </b>
An ADC (Analog to Digital converter), takes in an analog signal and converts into digital and feeds it to your digital system (computer, calculator, watch) which manipulates the data and produces an output through the DAC (Digital to Analog converter).

The digital system is made of sub systems, modules, basic units (Logic Gates), circuits (transistirs, capacitors etc).
